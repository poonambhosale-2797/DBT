2) Which subdivision of SQL is used to insert values in tables?
ANS:-Data Definition Language (DDL):-CREATE,DROP,ALTER
     Data Manipulation Language(DML):-INSERT,DELETE,UPDATE
     Data Control Language (DCL):-GRANT,REVOKE
     Data Query Language(DQL):-SELECT
     Data Transaction Language/Transaction Control Language:-Commit,Rollback,Savepoint
    DDL:-Rename,Truncate
